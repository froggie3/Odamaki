# Figures required

- odamaki.png

- default.png

- empty-thumbnail.png


## repository-ogp.png

It is displayed in a top README.md, and its size is no matter, but it should be sized in 1280x640 or made 1:0.525 in ratio.


## empty-thumbnail.png

It is displayed in place of normal photograph if a post has no thumbnails, it should be resized in 1200x630 or made 1:0.525 in ratio.


## default.png

It is displayed no matter where you are in the website apart from entry pages. It should be resized in 1200x630 or made 1:0.525 in ratio.
